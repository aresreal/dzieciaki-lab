# Małe Zabawy

Katalog kolorowych miniapek dla dzieci. Każda aplikacja ma własny folder i jest dostępna ze strony głównej.

## Dostępne projekty

- [Kalkulator Six Seven](kalkulator-six-seven/)
- [Planownia](generator-planow/)

## Struktura

```text
index.html                 # strona główna
kalkulator-six-seven/     # kalkulator
generator-planow/         # generator planów dla dzieci
```
